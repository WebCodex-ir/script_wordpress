# script_wordpress

\# پروژه نصب و تنظیم خودکار سرور



\## گام‌ها:



1\. اسکریپت `install.sh` را دانلود و اجرا کنید.

2\. اسکریپت‌ها به ترتیب به صورت خودکار نصب و پیکربندی می‌شوند:

&nbsp;   - Apache

&nbsp;   - WordPress

&nbsp;   - MySQL

&nbsp;   - FTP

&nbsp;   - SSL (Let's Encrypt)

3\. پس از اتمام، سایت شما آماده است.



\## توضیحات



\- فایل‌های کانفیگ Apache به‌طور خودکار ساخته و تنظیم می‌شوند.

\- MySQL به‌طور خودکار برای وردپرس تنظیم می‌شود.

\- SSL به‌طور خودکار برای دامنه‌ها تنظیم و تمدید می‌شود.
